# Sadaqah Jariyah for Ashpak Momin

## Documentation

### Features List
- Simple user interface for donating Sadaqah Jariyah.
- Track donations and their impact.
- Share your donation stories with the community.
- Get reminders for regular donations.
- Access educational content regarding Sadaqah Jariyah.

### Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/farhanamomin/sadqa-e-jariya.git
   ```
2. Navigate into the project directory:
   ```bash
   cd sadqa-e-jariya
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

### Usage Guide
- Open the application and create an account.
- Browse through different Sadaqah Jariyah opportunities.
- Select an option and follow the prompts to donate.

### App Screenshots Description
- **Home Screen:** Shows available Sadaqah Jariyah options.
- **Donation History:** Displays past donations and their impact.
- **Education Center:** Contains articles and videos on Sadaqah Jariyah.
